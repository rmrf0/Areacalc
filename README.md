# Areacalc
使用Qt6.8.1开发

IDE:Qt Creator 15.0.0

目前只支持windows，windows10测试可用，其他未知

## 兼容性
### Windows
|   系统名称   |   能否运行   |
|-------------|---------------|
| Windows 11 64位 | 能 |
| Windows 10 64位 | 能 |
| Windows 10 32位 | 不能 |
| Windows 7 64位 | 能 |
| Windows 7 32位 | 不能 |

## 功能如下：
- 1.圆环面积计算
- 2.三角形面积计算(已知三边的边长)
- 3.圆柱表面积计算
- 4.圆柱体积计算

- _后续还会持续更新,如果有bug或建议请在上方点击Issues发送_
