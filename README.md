# Areacalc
作者:rmrf0  
使用Qt6.8.1开发  
IDE:Qt Creator 15.0.0
## 兼容性
### Windows
|   系统名称   |   能否运行   |
|-------------|---------------|
| Windows 11 64位 | 能 |
| Windows 10 64位 | 能 |
| Windows 10 32位 | 不能 |
| Windows 7 64位 | 能 |
| Windows 7 32位 | 不能 |  
 
_Windows 8.x , Windows xp , Windows 98 ,Windows 95 和 Windows server没有测试，不知道能不能运行_
## 功能如下：
- 1.圆环面积计算
- 2.三角形面积计算(已知三边的边长)
- 3.圆柱表面积计算
- 4.圆柱体积计算
